# Privacy Policy — Grid Lens

**Last updated:** 2026-04-09

## Data Collection

Grid Lens does not collect, store, or transmit any user data. All processing happens locally in your browser.

## What Grid Lens Does

- Injects overlay scripts (pixel grid, rulers, element inspection) into the active tab when you click the extension
- Reads computed styles and CSS rules from page elements for design extraction
- Saves your preferences (toggle states, theme, grid size) locally using chrome.storage.local

## What Grid Lens Does NOT Do

- Does not collect personal information
- Does not track browsing history
- Does not make any network requests
- Does not use analytics or telemetry
- Does not share any data with third parties

## Permissions

- **activeTab**: Access the current page only when you click the extension
- **storage**: Save your preferences locally on your device
- **scripting**: Inject overlay and extraction scripts into the active tab
- **sidePanel**: Optional persistent panel alongside your browser
- **contextMenus**: Right-click menu on the extension icon
